\# 🛡️ Sentinel AI – Real-Time Cyber Fraud Prevention System



Sentinel AI is a real-time cybersecurity system that detects and blocks phishing, scam, and malicious websites \*\*before they cause damage\*\*.



\---



\## 🚀 Features



\* 🔍 Real-time URL monitoring (Chrome Extension)

\* 🧠 AI-based risk scoring system

\* ⚠️ Automatic blocking of dangerous websites

\* 📊 Explainable AI (shows reasons for detection)

\* 🔗 Backend + AI microservice architecture



\---



\## 🏗️ Architecture



```

Chrome Extension → Node.js Backend → Python AI Service

```



\---



\## 🧠 Detection Logic



\* Suspicious keywords (login, bank, verify)

\* Fake domains (g00gle, paypaI)

\* URL patterns (-, @, long URLs)



\---



\## ⚙️ Tech Stack



\* Frontend: Chrome Extension (JavaScript)

\* Backend: Node.js (Express)

\* AI Service: Python (Flask)

\* Communication: REST APIs



\---



\## 🛠️ Setup Instructions



\### 1. Start AI Service



```

cd ai-service

venv\\Scripts\\activate

python app.py

```



\---



\### 2. Start Backend



```

cd backend

npm install

node index.js

```



\---



\### 3. Load Chrome Extension



\* Go to `chrome://extensions/`

\* Enable Developer Mode

\* Click \*\*Load Unpacked\*\*

\* Select `/extension` folder
